This is a small automation project for my application for a QA role in Globe Telecom.
This project features a single test file that will execute an end-to-end test for Spree Commerce from account registration until a successful order placement. 
Along with this are simple CI/CD workflows - one which will run automatically upon pushing any changes to the repo, and the other which will run on demand with a configurable field.
